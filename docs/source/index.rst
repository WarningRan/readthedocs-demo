Welcome to DrCCTProf!
*********************
    
DrCCTProf is a fine-grained call path profiling framework for binaries running on ARM architectures, to obtain full calling context at any and every machine instruction, to associate any instruction with source code along the call path, and associate memory access instructions back to the constituent data object.

DrCCTProf provides a set of APIs to client tools. It is divided into the following sections:
